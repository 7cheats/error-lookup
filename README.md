  # 7Cheats Staff Error Lookup

  ## Client Issues

  ### General Client Connection Issues

  * Ask the user to use a VPN

    * Recommended: **Proton VPN**
    * ⚠️ **Cloudflare Warp does NOT work**

  ---

  ### Failed to Load Cheat

  **Check the executable name:**

  * If it looks like a long random string (e.g. `dasdhasudy272t7t136t2.exe`):

    * ➤ Have them **restart the cheat PC**

  * If it does NOT:

    * ➤ Ensure **all antivirus software is fully disabled**

  ---

  ## R6 - DMA Issues

  ### Unable to Connect to DMA

  * Likely hardware connection issue:

    * ➤ Have them **cold boot**
    * ➤ Try a **different cable**
    * ➤ Try a **different PCIe slot**

  ---

  ### Failed to Find Required Library

  * Missing dependencies:

    * ➤ Install required libraries (e.g. **dxwebsetup**)

  ---

  ### Unrecoverable Error Encountered (`XXX.cc:X`)

  * ➤ Have them **restart the game**
  * If error persists:

    * ➤ Request **QB upload**
    * ➤ ⚠️ Record exact error (e.g. `component.cc:91`)

  ---

  ### Authentication Failure

  * ➤ Have them use **Proton VPN**

  ---

  ### Unable to Complete Caching

  * Cause: Game open too long

    * ➤ Have them **restart the game**

  ---

  ### Game Not Running / Failed to Attach to Game

  * ➤ Ensure cheat is launched from **main menu**

  ---

  ### Unable to Initialize Threading

  * ➤ Unplug DMA device
  * ➤ Wait **30 seconds**
  * ➤ Plug back in

  If issue persists:

  * ➤ Ask for:

    * Exact DMA card model
    * Firmware being used
  * ➤ Escalate to developer

  ---
